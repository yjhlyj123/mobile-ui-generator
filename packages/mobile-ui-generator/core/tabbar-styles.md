# Tabbar 页面风格预设库

> 一级页面（Tabbar 页面）的风格直接决定用户对应用的第一印象。
> 本文件提供 4 种差异化风格预设，Agent 在检测到 Tabbar 页面需求时，
> 应从中选择或组合至少 3 种方案呈现给用户选择。
>
> **所有风格均须适配微信小程序胶囊区域**，详见 `page-hierarchy.md` § 3.0。
> **所有风格头部均须使用品牌化设计元素**（非纯色），详见 `page-hierarchy.md` § 3.1。

---

## 风格 A：沉浸式头部

### 设计理念
头部使用品牌渐变色占据 30%-40% 首屏，内容卡片上浮叠加，形成沉浸感。

### 布局示意

```
┌───────────────────────────────────────────┐
│ 状态栏                                     │
├─────────────────────────────┬─────────────┤
│  "你好，张三"                │  [ 胶囊 ]   │ ← 避开胶囊区域
├─────────────────────────────┴─────────────┤
│     🌊 渐变背景 + 几何装饰/纹理            │
│         🔍 搜索框（透明底）                 │
│                          ◯ (0.08)         │ ← 装饰元素
│                      ◯      (0.05)        │
├─────┬─────┬─────┬─────┬─────┬─────────────┤
│ 📋  │ 📊  │ ✅  │ 📝  │ 📁  │             │ ← 金刚区（悬浮卡片）
│审批  │统计  │签章 │合同  │文档  │             │
└─────┴─────┴─────┴─────┴─────┴─────────────┘
┌───────────────────────────────────────────┐
│ 📌 待办事项                                │
│ ├─ 合同审批 - 张三 - 2h前                  │
│ ├─ 用章申请 - 李四 - 3h前                  │
│ └─ 查看更多 →                              │
└───────────────────────────────────────────┘
┌───────────────────────────────────────────┐
│ 📊 数据概览                                │
│ ┌──────┐ ┌──────┐ ┌──────┐               │
│ │ 待办  │ │ 进行  │ │ 完成  │               │
│ │  12  │ │  8   │ │  45  │               │
│ └──────┘ └──────┘ └──────┘               │
└───────────────────────────────────────────┘
```

### 关键特征
- 头部高度：250rpx+（不含状态栏和导航栏）
- 渐变方向：135° 或 180°
- 金刚区悬浮在渐变与内容区交界处
- 白色文字 + 半透明搜索框
- 内容卡片使用 `$radius-xxl` 大圆角

### 背景设计元素
- 推荐风格 A（几何装饰）：右下角 1-2 个低透明度圆形
- 也可用风格 B（纹理）：整体覆盖微弱点状纹理

### 微信小程序适配
- 自定义导航栏叠加在渐变背景上，问候语和搜索框在导航栏下方
- 问候语左对齐，右侧留出胶囊宽度 + 12px

### 适用场景
工作台首页、OA 主页、综合管理类应用

## 风格 B：卡片金刚区

### 设计理念
以功能入口为核心，金刚区占据首屏核心位置，弱化头部，强调操作效率。

### 布局示意

```
┌───────────────────────────────────────────┐
│ 状态栏                                     │
├─────────────────────────────┬─────────────┤
│ Logo  "工作台"       🔔 📋   │  [ 胶囊 ]   │ ← 简洁导航栏
├─────────────────────────────┴─────────────┤
│  渐变背景条 + 装饰元素（高度较低 ~120rpx）   │
│  ┌──────────────────────────────────────┐ │
│  │  ┌──────┐ ┌──────┐ ┌──────┐         │ │
│  │  │ 🔵   │ │ 🟢   │ │ 🟠   │         │ │ ← 大图标金刚区
│  │  │ 电子  │ │ 合同  │ │ 审批  │         │ │
│  │  │ 签章  │ │ 管理  │ │ 中心  │         │ │
│  │  ├──────┤ ├──────┤ ├──────┤         │ │
│  │  │ 🔴   │ │ 🟣   │ │ ⚫   │         │ │
│  │  │ 工资  │ │ 考勤  │ │ 更多  │         │ │
│  │  │ 单    │ │ 打卡  │ │      │         │ │
│  │  └──────┘ └──────┘ └──────┘         │ │
│  └──────────────────────────────────────┘ │
├───────────────────────────────────────────┤
│ ┌──────────┐ ┌────────────────┐           │
│ │ 待审批    │ │ 公告通知        │           │
│ │   12     │ │ 新通知 3 条     │           │
│ │ 本周 +3  │ │ 查看全部 →      │           │
│ └──────────┘ └────────────────┘           │
├───────────────────────────────────────────┤
│ 📋 最近使用                                │
│ ├─ 合同签署 - 10:30                        │
│ ├─ 印章管理 - 昨天                          │
│ └─ 更多 →                                  │
└───────────────────────────────────────────┘
```

### 关键特征
- 导航栏简洁，仅 Logo + 标题 + 操作图标
- 金刚区图标使用品牌色渐变圆形底色
- 图标卡片较大（80rpx+），3 列布局
- 头部渐变区域较窄，金刚区卡片悬浮叠加
- 下方紧跟统计卡片（2 列网格）
- 页面底色 `$color-bg-page`

### 背景设计元素
- 头部渐变条高度较低（~120rpx），使用品牌渐变色
- 推荐风格 A（几何装饰）：渐变区底部带弧形切割线
- 金刚区卡片悬浮叠加在渐变与白底交界处

### 微信小程序适配
- 导航栏使用简洁模式（Logo + 标题），右侧图标避开胶囊
- 渐变背景从状态栏贯穿到金刚区上方

### 适用场景
功能导向型应用、工具类小程序、B 端管理入口

## 风格 C：数据看板式

### 设计理念
以数据为核心，首屏展示关键指标和趋势，适合需要数据驱动决策的场景。

### 布局示意

```
┌───────────────────────────────────────────┐
│ 状态栏                                     │
├─────────────────────────────┬─────────────┤
│  📊 数据概览      筛选 ▼     │  [ 胶囊 ]   │
├─────────────────────────────┴─────────────┤
│  渐变背景 + 几何装饰                        │
│ ┌───────────────────────────────────────┐ │
│ │   💰 总金额                            │ │
│ │   ¥ 1,234,567                         │ │
│ │   ↑ 12.5% 较上月                       │ │
│ └───────────────────────────────────────┘ │
│ ┌──────┐ ┌──────┐ ┌──────┐               │
│ │ 待签  │ │ 执行  │ │ 完成  │               │
│ │  23  │ │  15  │ │  89  │               │
│ │ +5   │ │ -2   │ │ +12  │               │
│ └──────┘ └──────┘ └──────┘               │
├───────────────────────────────────────────┤
│  [本周] [本月] [本季] [自定义]              │ ← 时间筛选 Filter Pill
├───────────────────────────────────────────┤
│ ┌───────────────────────────────────────┐ │
│ │   📈 趋势图                            │ │
│ │   ▁▂▄▇█▆▃▂▅▇                          │ │
│ └───────────────────────────────────────┘ │
├───────────────────────────────────────────┤
│ 📋 TOP 5 待办                              │
│ 1. 紧急合同审批 - 2h内                      │
│ 2. 客户签章确认 - 今天                       │
│ ...                                        │
└───────────────────────────────────────────┘
```

### 关键特征
- 核心指标卡片使用品牌色底或渐变底
- 数字使用超大字号 `$font-size-hero`+
- 趋势标识使用功能色（涨绿跌红）
- 时间筛选器使用 Filter Pill 样式
- 可选内嵌图表组件

### 背景设计元素
- 推荐风格 A（几何装饰）：头部渐变 + 圆形装饰，避免纹理干扰数据阅读
- 核心指标卡片可带品牌色半透明底色
- 指标卡片悬浮在渐变区交界处

### 微信小程序适配
- 标题 + 筛选入口在导航栏内，避开胶囊
- 渐变背景贯穿到核心指标卡片区域

### 适用场景
管理者视角、数据驾驶舱、业绩看板、统计报表入口

## 风格 D：图文并茂式

### 设计理念
用视觉插图和温暖的设计语言降低工具感，提升 C 端用户的体验亲和力。

### 布局示意

├─────────────────────────────┬─────────────┤
│  "早上好！"          🔔      │  [ 胶囊 ]   │
├─────────────────────────────┴─────────────┤
│  🎨 品牌渐变背景 + 纹理装饰                 │
│      ┌────────────────┐                    │
│      │   🏠 品牌插图   │                    │
│      │  欢迎回来！     │                    │
│      └────────────────┘                    │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐     │
│  │ 🏃‍♂️  │ │ 📚   │ │ 💊   │ │ 📋   │     │
│  │ 运动  │ │ 学习  │ │ 用药  │ │ 记录  │     │
│  └──────┘ └──────┘ └──────┘ └──────┘     │
├───────────────────────────────────────────┤
│ 🌟 今日推荐                                │
│ ┌───────────────────────────────────────┐ │
│ │ 📸 大图卡片                            │ │
│ │ 标题文字                               │ │

│ └───────────────────────────────────────┘ │

│ │ 📸 小图       │ │ 📸 小图       │         │
│ │ 标题         │ │ 标题         │         │
│ └──────────────┘ └──────────────┘         │
├───────────────────────────────────────────┤

│  [卡片1] [卡片2] [卡片3]                    │

### 关键特征
- 头部使用品牌插图（建议 SVG 或矢量风格）
- 圆角更大 `$radius-xxl`+
- 卡片可带品牌色浅底色
- 可使用横向滚动展示推荐内容
- 整体色调更温暖、更 C 端

### 背景设计元素
- 推荐风格 B（微弱纹理）：渐变 + 点状纹理，增加温暖质感
- 品牌插图叠加在渐变区内（插图建议用 SVG，避免大图片加载慢）
- 也可用风格 A + 插图组合

### 微信小程序适配
- 问候语 + 通知图标在导航栏内，避开胶囊
- 渐变背景贯穿到金刚区下方

### 适用场景
C 端消费类、教育学习、生活服务、健康医疗

## Agent 使用指南

### 何时触发

当以下信号之一为 true 时，Agent 应视当前页面为 Tabbar 页面：
- 用户明确提到"首页"、"主页"、"Tabbar"、"底部导航"
- 页面配置中包含 `tabbar` 或 `tabBar`
- 使用了 `/mug:tabbar` 命令

### 方案呈现

1. 从上述 4 种风格中选择至少 3 种适合用户场景的方案
2. 每种方案说明：
   - 风格名称和一句话定位
   - 布局 ASCII 示意图
   - 选用的背景风格（A 几何装饰 / B 微弱纹理）及原因
   - 3 个适用场景关键词
   - 与其他方案的核心差异
3. 等待用户选择后再进入实现

### 组合使用

不同 Tabbar 页面可以使用不同风格：
- "首页" → 沉浸式头部 或 数据看板
- "工作台" → 卡片金刚区
- "我的" → 沉浸式头部（带头像和信息）
- "发现" → 图文并茂式
