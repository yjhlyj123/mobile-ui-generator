# Codex 适配器：Pencil Prompt 模板

规范来源：`../../core/pencil-prompt-template.md`

## 标准模板

```md
# 页面信息
- 页面类型：
- 业务目标：
- 目标用户：
- 业务场景：

# 核心模块
- 模块 1：
- 模块 2：
- 模块 3：

# 优先级
- 首屏必须表达：
- 次级信息：
- 可省略信息：

# 风格约束
- B端企业页面
- 理性蓝灰
- 信息克制
- 弱装饰
- 不要营销式头图
- 不要冗余描述文案

# 技术与落地约束
- uni-app
- Vue 3
- UnoCSS
- wot-design-uni

# 参考信息
- 参考稿 / `.pen` / 截图：
- 需要保留的元素：
- 需要避免的元素：

# 输出要求
- 给出页面结构
- 给出区块层级
- 给出首屏重点
- 给出自定义导航建议（如需要）
- 给出状态与卡片的视觉方向
```

## 重构模式模板（用户提供参考页面并要求重构时使用）

本模板用于 **3 个方向**，每个方向独立填写，替换 `[方向名]` 并逐项填充。

```md
# 重构方向：[方向名]

## 方向策略
- 核心差异（与参考页的区别）：
- 布局策略：
- 信息层级重心：

## 从参考页提取的内容（仅业务内容，不含视觉）
- 模块列表：
- 数据字段：
- 业务状态：
- 操作入口：

## 本方向的设计指令
- 首屏重点：
- 区块划分：
- 状态呈现方式：
- 操作位置与权重：

## 明确禁止复现参考页的以下元素
- （列出参考页中不应被照搬的布局、卡片形态、颜色用法、信息分组等）

## 风格约束
- B端企业页面
- 理性蓝灰
- 信息克制
- 弱装饰
- 不要营销式头图
- 不要冗余描述文案

## 技术与落地约束
- uni-app
- Vue 3
- UnoCSS
- wot-design-uni

## 输出要求
- 给出页面结构
- 给出区块层级
- 给出首屏重点
- 给出与其他方向的关键视觉差异说明
```
